## 🌐 API Information
Base URL: https://prog2005.it.scu.edu.au/ArtGalley
- `GET /` — Get all items
- `GET /name` — Get item by name
- `POST /` — Add new item
- `PUT /name` — Update item by name
- `DELETE /name` — Delete item by name
